# Ruangguru Enterprise Assignment | Application
[Demo](https://rg-package.vercel.app/)

## How to run the project
1. Clone this repository
2. Copy `.env.example` file to `.env`.
3. Edit `REACT_APP_API_ENDPOINT` with API's endpoint.
4. Run this command:
```
yarn install
yarn start
```
or
```
npm install
npm run start
```
